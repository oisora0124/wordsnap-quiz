📖 ツールの使い方
  1. 取り込む：英単語帳の写真（OCR）や、テキストの貼り付けで単語を取り込みます。1行に「英単語＋意味」（例: apple りんご）。
  2. 確認：取り込んだ候補をチェックして「保存」。保存先の単語帳も選べます。
  3. クイズ：保存した単語で意味の4択クイズ。出題する単語帳と範囲（番号）を選べます（「範囲」ボタンで指定。未指定は全部）。
  4. 保存した単語：一覧・検索（単語/意味の両方向）・番号で範囲削除。JSONを書き出すでバックアップできます。
  5. 同期・引き継ぎ：データは自動でサーバー保存。別端末で使うには「個人リンクをコピー」して、その端末でリンクを開きます。端末で内容がずれたら「端末ごとに単語が違うとき」で手動修復。

💡 Safariで開いて「ホーム画面に追加」しておくと、データが消えにくく、ワンタップで起動できます。

🔄 データの保存・引き継ぎ

追加した単語とクイズの記録は、自動でサーバーにも保存されます。あなた専用のキーで保管されるので、他の人のデータと混ざりません。

  1. ホーム画面に追加して使う：このページを「ホーム画面に追加」し、そのアイコンから開いてください。ブラウザの保存が消えても、開き直せばサーバーから自動で元に戻ります。
  2. 別の端末で続ける：「個人リンクをコピー」して別の端末で開くと、同じデータを引き継げます（このリンクは自分だけの秘密にしてください）。

サーバー保存の状態をここに表示します。

端末ごとに単語が違うとき（手動で揃える）

同じ個人リンクなのに端末で内容が食い違うときは、どちらかを「正しい方」に決めて揃えます。 （例：片方で削除した単語が、もう片方から戻ってきてしまう場合など）

手順：①正しい端末で「この端末を正にする」を押す → ②もう一方の端末で「他の端末に合わせる」を押す。 どちらも上書きなので、押す前に確認が出ます。
