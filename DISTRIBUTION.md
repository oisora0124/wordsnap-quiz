# WordBank 配布ガイド

公開URL: https://wordbank.pages.dev

このアプリはサーバーにアカウントを持ちません。単語と学習履歴は**各自のブラウザ**に保存され、
公開版では自動発行される個人キーに紐づけてWordBankの同期サーバーにも保存されます。
配布時は、この文書の「利用者への説明」節をそのまま渡せば足ります。

---

## 1. APIキーについて

### 結論：キーは各自で取得してもらう。設定しなくても使える。

アプリはAIキーが**なくても動きます**。あるとクイズの質が上がる、という位置づけです。

| | キーなし | キーあり |
|---|---|---|
| 単語の出題 | ○ | ○ |
| 例文モード（空所補充） | 約88%の語で出題できる（実測） | ほぼ全語 |
| 例文の質 | 辞書の実例。一部は定義の断片 | 語義に合わせてAIが作成 |
| 誤答選択肢 | 辞書由来の派生形＋登録済みの単語 | AIが文脈に合わせて作成し、自己検証も行う |
| 通信 | 辞書API・Datamuse（キー不要・無料） | 上記＋AI |

**キーなしの実測値**（内蔵サンプル300語から120語を抽出、dictionaryapi.dev）:

| | 結果 |
|---|---|
| 空所補充にできた | 105/120 = **87.5%** |
| うち文として自然 | 98/120 = 81.7% |
| 定義の断片（例: `moral absolutes`） | 6.7%（文らしい例文を優先する処理により 18.5% から改善） |
| 通常の単語出題に回る | 15/120 = 12.5% |
| 辞書APIの応答時間 | 中央値 24ms / p90 74ms |

キーなしでも大半の語が空所補充で出題できます。「キーがないと使えない」わけではありません。

キーは**利用者自身のブラウザにのみ**保存されます（既定はタブを閉じると消える。「この端末に保存」を
選んだときだけ残る）。配布者のキーが共有されることはありません。

### 無料枠は足りるか：足ります

アプリは**1単語につきAI呼び出し1回**です（例文・誤答3つ・自己検証を1回の応答でまとめて取得）。
一度作った内容は端末にキャッシュされ、同じ語で再度呼ぶことはありません。

| プロバイダ | アプリが使うモデル | 無料枠（2026年7月時点） |
|---|---|---|
| Google Gemini | gemini-3.6-flash（キーで使えない場合は 3.5 / 2.5 Flash へ自動フォールバック） | 約 10回/分・1,000〜1,500回/日 |
| Groq | qwen/qwen3.6-27b | 30回/分・1,000回/日 |

内蔵サンプル「大学受験標準レベル単語300」を全部やっても**300回**なので、1日の枠に十分収まります。

> **注意**: Googleは現在レート制限の具体的数値を公式ドキュメントに掲載しておらず、
> [AI Studio のダッシュボード](https://aistudio.google.com/rate-limit)で確認する方式に変わりました。
> 上表のGeminiの数値は第三者情報に基づく目安です。正確な値は各自のAI Studioで確認してください。
> Groqの数値は[公式ドキュメント](https://console.groq.com/docs/rate-limits)記載のものです。

### 実際に効いてくるのは「1日」より「1分」の上限

Geminiの無料枠は**10回/分**です。アプリは次の問題を先読みするため、テンポよく解くと
1分あたり10語を超えることがあります。この場合そのリクエストは失敗しますが、
**一時的な障害として扱い、キャッシュには残しません**（時間をおけば同じ語でAI生成をやり直します）。
その問題だけ辞書由来の出題になり、学習は止まりません。

より速いテンポで使うなら、**Groq（30回/分）**のほうが余裕があります。

### キーの取得先

- Gemini: https://aistudio.google.com/apikey
- Groq: https://console.groq.com/keys

## 2. 個人キー（`?w=`）の扱い — 配布時に必ず伝えること

公開版を初めて開くと、ブラウザの安全な乱数機能で個人キーを自動発行し、URLは次の形になります。
同じURLを別端末で開くと、同じ保存先を利用できます。

```
https://wordbank.pages.dev/?w=ws_xxxxxxxxxxxxxxxx
```

**このURLを知っている人は、その学習データを読み書きできます。パスワードはありません。**

- ✅ 自分の別端末（スマホ・PC）に送るのはOK
- ❌ SNS・共有チャット・スクリーンショットに載せるのはNG
- ❌ 「このアプリいいよ」と人に薦めるとき、`?w=` 付きのURLを送らない

薦めるときは `https://wordbank.pages.dev`（クエリなし）を送ってください。
アプリ内の共有ボタンはクエリを除いたURLだけを共有するようになっており、設定画面でも
このキーは「秘密」と明示しています。ただし**アドレスバーからURLをコピーすると
キーごとコピーされる**ため、配布時の口頭説明でも触れてください。

公開版では個人キーの保存先へ自動同期します。外部サービスへの送信内容は設定画面の
「データの保存場所と削除について」で確認できます。サーバーへ保存したくない用途では、
「ツールを保存」で作った単一HTMLを端末内で使えますが、`file://` ではブラウザにより
保存が制限される場合があるため、JSONバックアップも併用してください。

## 3. 利用者への説明（そのまま渡せる文面）

> **WordBank — 英単語アプリ**
> https://wordbank.pages.dev
>
> ブラウザで開くだけで使えます。インストールも登録も不要です。
> 「STEP1 取り込む」の**大学受験標準レベル単語300**ボタンから、すぐ始められます。
> 自分の単語帳を写真から取り込むこともできます。
>
> **AIキー（任意）**
> 設定画面で無料のAPIキーを入れると、例文の空所補充クイズが全単語で使えるようになります。
> 入れなくても普通に使えます。キーは自分のブラウザにだけ保存され、他の人には渡りません。
> - Gemini（無料）: https://aistudio.google.com/apikey
> - Groq（無料）: https://console.groq.com/keys
>
> **データについて**
> 単語と成績は自分の端末に保存され、公開版では自動発行された個人キーに紐づけて
> WordBankの同期サーバーにも保存されます。設定の「JSONバックアップ」から
> 書き出しておくと、誤操作やキー紛失から戻しやすくなります。
>
> **同期キーを使うときの注意**
> 複数端末で同期すると、URLに `?w=` で始まる長い文字列が付きます。
> **これはパスワード代わりです。人に送らないでください。**
> アプリを人に薦めるときは `https://wordbank.pages.dev` だけを送ってください。

## 4. 運用側で見ておくもの

- **Cloudflare D1（同期用）** — 公開版は個人キーを自動発行して同期するため、利用者が増えたら
  読み取り/書き込みの無料枠を確認する。非表示タブでは定期ポーリングを止め、表示へ戻ったときに再開する。
- **辞書API（dictionaryapi.dev）のレート制限** — 短時間に連続リクエストすると 429 を返します
  （測定時、5並列で300件送ったところ275件が拒否されました）。アプリは1問ずつ＋次問の先読みしか
  投げないため通常は問題になりませんが、利用者が増えても各自のブラウザから直接叩く構成なので
  配布者側の負荷にはなりません。
- **実機での通し確認（AIキーあり）** — キーなしの経路は上記のとおり実測済み。
  AI経路は実際のキーが要るため未計測。配布前に数十語を回して生成品質と体感速度を見ておくと安心です。

> Netlify版バックエンドは削除済み（デプロイが停止しており、非原子的CAS・圧縮非対応のままだったため）。
> 必要になれば git 履歴から復元できます。
